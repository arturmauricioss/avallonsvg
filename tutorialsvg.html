Editor e Compilador SVG

O coração deste tutorial é o editor e compilador SVG. Com esta ferramenta, você pode criar e editar facilmente seus próprios gráficos vetoriais. O editor fornece uma experiência de edição de código avançada com destaque de sintaxe, números de linha e muito mais alimentado pelo estilizador CodeMirror, uma biblioteca JavaScript

Visualização em Tempo Real

À medida que você edita o código SVG no editor, a visualização em tempo real é atualizada automaticamente no painel de visualização. Isso permite que você veja imediatamente as mudanças que está fazendo no seu gráfico.

Carregamento de Arquivos SVG

Além de criar SVGs do zero, você também pode carregar arquivos SVG existentes para editar aqui. Basta clicar na área de visualização e selecionar o arquivo SVG desejado em seu computador.

Compilação para SVG e PNG

Uma vez satisfeito com o seu gráfico, você pode compilar o código SVG em um arquivo SVG para download ou em um arquivo PNG para uso em outros projetos. Os botões "SVG" e "PNG" facilitam esse processo.

Responsivo e Pronto para Dispositivos Móveis

Avallon SVG é projetado para funcionar em uma variedade de dispositivos, desde desktops até dispositivos móveis. Ele usa técnicas de design responsivo para garantir uma experiência consistente em diferentes tamanhos de tela.

Seguro e de Código Aberto

É importante dizer que Avallon SVG apenas instrui seu computador no que ele deve fazer de acordo com o seu requerimento, portanto nada do que você faça sairá do seu computador pois todo código de script acontece bem ai, o projeto é de código aberto e está disponível em meu Github.
